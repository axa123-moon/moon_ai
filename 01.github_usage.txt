1.현재 상태 확인
git git

2.파일 가져오기
git pull origin master

3.파일 추가하기
git add .
git commit -m "message"
git push origin +master

4.다른 사람 git source 가져오기
  1) 신규 폴더 생성
  2) 마우스 우측버튼을 이용하여 "Git Bash Here" 선택
  3) 아래 명령어 실행
     가) 해당 폴더 github 초기화
          git init   
     나) 원하는 github연결
          git remote add origin https://github.com/axa123-moon/moon_ai
     다) 해당 github에서 파일 가져오기
          git pull origin master
